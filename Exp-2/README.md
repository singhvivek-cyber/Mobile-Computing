# Cellular Frequency Reuse Game

An interactive, educational game built with **Jetpack Compose** to visualize and master the concept of frequency reuse in cellular networks.

## Features
- **Dynamic Hexagonal Grid:** A custom-drawn honeycomb layout using axial coordinates.
- **Interactive Logic:** Randomized i and j parameters to calculate Cluster Size (N).
- **Material 3 Design:** Sleek dark-themed UI with Gold and Teal color accents.
- **Educational Value:** Helps users visualize co-channel interference and safe frequency distances.

## Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Modern Declarative UI)
- **Minimum SDK:** API 24+
- **Compiler:** Compose Compiler 1.5.14 (Optimized for Kotlin 1.9.24)
