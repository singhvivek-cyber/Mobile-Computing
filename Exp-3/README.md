## Features
- **Smooth Stroke Engine:** Uses `StrokeCap.Round` for professional, rounded line edges.
- **Glassmorphic Toolbar:** A floating, semi-transparent tool palette with shadow effects.
- **Color Palette:** Quick-switch between Teal, Gold, Red, and White.
- **Undo & Clear:** One-tap icons to remove the last stroke or wipe the entire canvas.
- **Modern Background:** Subtle radial gradient for a premium "Dark Mode" feel.

## Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Minimum SDK:** API 24+
- **Compiler Version:** 1.5.14 (Compatible with Kotlin 1.9.24)
